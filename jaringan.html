Pengertian Malware Phishing DDoS Pencegahan

1. Pengertian Serangan Jaringan

Serangan jaringan adalah usaha yang dilakukan oleh pihak tidak bertanggung jawab untuk mengganggu, mencuri, atau merusak sistem jaringan komputer.

Serangan ini dapat menyebabkan data bocor, layanan down, dan kerugian besar.

2. Malware

Malware adalah software berbahaya seperti virus, worm, dan trojan.

  • Virus: menyebar lewat file
  • Worm: menyebar lewat jaringan
  • Trojan: menyamar sebagai aplikasi normal

Dampak: data rusak, sistem lambat, kehilangan file.

3. Phishing

Phishing adalah penipuan untuk mencuri data pribadi seperti password dan akun.

Biasanya lewat:

  • Email palsu
  • Website tiruan
  • Pesan WhatsApp/DM

Dampak: akun dibajak, data dicuri.

4. DDoS (Distributed Denial of Service)

DDoS adalah serangan yang membuat server tidak bisa diakses karena terlalu banyak permintaan.

Dampak: website down, layanan tidak bisa digunakan.

Catatan: Serangan ini ilegal dan dilarang hukum.

5. Cara Pencegahan Serangan Jaringan

  • Gunakan password yang kuat
  • Aktifkan firewall
  • Update sistem secara rutin
  • Gunakan antivirus
  • Jangan klik link sembarangan

Keamanan jaringan lebih penting daripada kemampuan menyerang.
