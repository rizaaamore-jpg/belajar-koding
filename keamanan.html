Keamanan Jaringan

Penjelasan: Keamanan jaringan adalah usaha melindungi jaringan dari akses ilegal dan kerusakan.

Contoh: WiFi sekolah dipakai orang luar tanpa izin.

Cara Mengatasi: Gunakan password WiFi, ganti secara berkala, aktifkan firewall.

2

Ancaman Jaringan

Penjelasan: Ancaman jaringan adalah gangguan yang bisa merusak atau mencuri data.

Contoh: Komputer tiba-tiba lambat karena program asing.

Cara Mengatasi: Update sistem, pasang antivirus, hindari software ilegal.

3

Malware

Penjelasan: Malware adalah program berbahaya yang merusak sistem atau mencuri data.

Contoh: HP error setelah install aplikasi bajakan.

Cara Mengatasi: Install aplikasi dari sumber resmi dan scan antivirus.

4

Virus Komputer

Penjelasan: Virus adalah malware yang menyebar dan merusak file.

Contoh: File tugas tiba-tiba rusak.

Cara Mengatasi: Antivirus aktif dan hindari flashdisk tidak dikenal.

5

Firewall

Penjelasan: Firewall menyaring data yang masuk dan keluar jaringan.

Contoh: Website berbahaya otomatis diblokir.

Cara Mengatasi: Aktifkan firewall di router dan komputer.

6

Enkripsi Data

Penjelasan: Enkripsi mengubah data menjadi kode agar tidak bisa dibaca.

Contoh: Password disimpan dalam bentuk acak.

Cara Mengatasi: Gunakan HTTPS dan aplikasi terpercaya.

7

Password Aman

Penjelasan: Password kuat sulit ditebak.

Contoh: "123456" sangat mudah dibobol.

Cara Mengatasi: Gunakan kombinasi huruf, angka, dan simbol.

8

Phishing

Penjelasan: Phishing adalah penipuan untuk mencuri data.

Contoh: Link hadiah palsu lewat WhatsApp.

Cara Mengatasi: Jangan klik link mencurigakan.

9

Keamanan WiFi

Penjelasan: WiFi perlu diamankan agar tidak disusupi.

Contoh: WiFi lemot karena dipakai orang asing.

Cara Mengatasi: Gunakan WPA2/WPA3 dan ganti password.

10

Etika & Kesadaran Digital

Penjelasan: Menggunakan teknologi secara bertanggung jawab.

Contoh: Tidak menyebarkan data pribadi orang lain.

Cara Mengatasi: Edukasi diri dan patuhi aturan.
